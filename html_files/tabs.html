To achieve this trick, I used CSS pesudo-classes ! Based on whether an input is "checked" or not, I can control whether that content will be displayed or not. That is, the content of all this webpage is in one html document. To verify this, you right click->source to see the HTML source code.

Extra

As a bonus, note how the cursor and color of the tabs change once they are hovered. This can be achieved with certain CSS psuedo-classes as well! Pseudo-classes are a way of applying styling only when certain conditions are met, which is a powerful tool you guys should get accustom to.
