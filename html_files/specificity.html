This page has two stylesheets linked to the page. Inspect element this page to see the stylesheets, and pay attention to which style ultimately ends up being applied. For instance, the header is colored by stylesheet 2.

Elements that are colored red are colored by the first stylesheet, elements colored blue are colored by the second stylesheet, and green if it is colored by inline CSS.
